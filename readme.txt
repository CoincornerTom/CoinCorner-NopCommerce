This is the official CoinCorner.com NopCommerce Payment module for integration with Bitcoin

== Description ==

The CoinCorner NopCommerce Payment module allows any site to accept payment in Bitcoin. Once connected via your API keys in your CoinCorner account, you are free to accept payments in Bitcoin and instantly be converted into GBP.

Features:

* Accept BTC payments
* Get the exact fiat amount back for your payments based on the current price at CoinCorner.com
* Low 1% fees for accepting payments.
* Easy to use, easy to integrate, easy for your customers.

*Please note you will need an account at CoinCorner.com to use this plugin*

Once installed, your site will simply have a Pay with Bitcoin option and will be ready to go. Yay!


== Installation ==

To install, simply add this plugin to your NopCommerce plugins and configure with your CoinCorner.com API keys. If Downloading from GitHub zip up the folder and the uploadeditems.json

1. Upload 'CoinCornerCheckout.zip' via the Admin panel -> Configuration -> Local Plugins. Click Upload plugin or theme then select the zip
2. Add your API keys at NopCommerce -> Configuration -> Local Plugins-> Click Configure for Bitcoin. If you do not have an account, you can sign up at CoinCorner.com
3. Activate the plugin through the 'Payment Methods' menu in NopCommerce Admin -> Payment Methods -> Edit and enable 'Is Active'
